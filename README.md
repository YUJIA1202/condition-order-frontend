# 条件单自动交易系统 — 前端

基于 React + Vite 的条件单监控与管理界面，已实盘运行。

## 技术栈

- React 18 / Vite
- WebSocket 实时行情接收

## 主要功能

- 股指价格条件单管理面板
- 量价均线条件单管理面板
- 实时行情监控与触发日志
- 市场情绪仪表盘
- 月历盈亏热力图
- 持仓展示

## 本地运行

1. 安装依赖：`npm install`
2. 确保后端服务已在 `http://localhost:8000` 运行
3. 启动：`npm run dev`
